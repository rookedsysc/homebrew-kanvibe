# Rookedsysc Kanvibe

## How do I install these formulae?

`brew install rookedsysc/kanvibe/<formula>`

Or `brew tap rookedsysc/kanvibe` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "rookedsysc/kanvibe"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
